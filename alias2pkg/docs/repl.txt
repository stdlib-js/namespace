
{{alias}}( alias )
    Returns the package name associated with a provided alias.

    Parameters
    ----------
    alias: string
        Alias.

    Returns
    -------
    out: string|null
        Package name.

    Examples
    --------
    > var v = {{alias}}( '{{alias:@stdlib/math/base/special/sin}}' )
    '@stdlib/math/base/special/sin'
